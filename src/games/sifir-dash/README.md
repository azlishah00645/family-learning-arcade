Year 3 Maths endless runner. Registers as `{ grade: "Year 3", subject: "maths" }`.

3-lane auto-runner: multiplication tables 2–9, division facts, add/subtract
within 1000. Combo multipliers, hearts, high scores, unlockable runners,
daily streak. Content: procedural generator in `/content/y3-math-engine.js`
plus optional fixed questions in `/content/y3-custom-questions.js`.
